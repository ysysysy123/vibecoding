## 💡 学习建议

1. **按顺序读**：从基础开始，循序渐进
2. **动手敲命令**：光看没用，跟着敲一遍
3. **用 practice-guide 里的练习**：每个练习都做一遍
4. **遇到问题先 git status**：90% 的疑惑看一下状态就明白了
